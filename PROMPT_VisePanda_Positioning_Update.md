覆盖从"想去的念头"到"在路上的每一分钟"的全链路，但分阶段交付：

### Phase 1 — 行前规划（当前重点）

| 环节 | 说明 |
|------|------|
| 前期咨询 | 什么时候去？去哪几个城市？玩几天？什么路线合理？ |
| 注意事项 | 签证类型、入境政策、手机卡、VPN、支付方式、文化差异提醒 |
| 定行程 | AI 生成完整行程单，含城市顺序、每天安排、推荐时长 |
| 订酒店 | 推荐适合外国人的酒店（接待资质、英文服务、位置）、比价/预订引导 |
| 订大交通 | 机票建议 + 高铁班次推荐（12306 外国护照购票指引） |

### Phase 1.5 — 行程中服务（行程确认后）

> 用户有了日程安排后，VisePanda 从"规划工具"变成"路上管家"

| 环节 | 说明 |
|------|------|
| 🆕 **翻译** | 实时翻译：拍照翻译菜单/路牌、语音翻译对话、常用中文短语速查（点菜、打车、问路） |
| 选餐厅 | 根据当前位置+口味偏好推荐餐厅，含外国人友好的判断（有无英文菜单、是否收外卡） |
| 选路线 | 城市内一日游路线建议，交通方式衔接（地铁、打车、步行） |
| 叫的士 | 滴滴/高德打车引导，上车给司机看中文地址，沟通模板 |
| 买团购 | 美团/大众点评团购券购买指引，外国人能用哪些 |

### Phase 2 — 行程后（日后技术成熟再做）

- 社区：游记分享、结伴同行
- 反馈：目的地评分、避坑提醒
- UGC 内容生态

## 翻译功能说明

翻译是 Phase 1.5 的核心能力，必须原生内置（不是跳转到第三方翻译App）：

| 翻译场景 | 实现方式 |
|----------|----------|
| 🏪 **拍照翻译** | 摄像头拍照 → OCR 识别中文 → 翻译成英文显示（菜单、路牌、告示、商品标签） |
| 🗣️ **语音翻译** | 按住说话 → 识别中文/英文 → 互译朗读（打车沟通、酒店前台、餐厅点菜） |
| 📝 **文本翻译** | 输入或粘贴中文 → 翻译成英文（或反向），含上下文优化 |
| 💬 **常用短语** | 分类短语库：点菜、打车、酒店入住、购物、问路、紧急情况，可发音 |
| 🧾 **实景翻译** | 摄像头实时取景，AR 叠加英文翻译（进阶功能） |

翻译数据用后即焚，不做长期存储。优先保证速度和离线可用性（基础短语包）。

## 产品定位更新要点

### 一句话产品
**VisePanda is your all-in-one China travel butler — from planning your first trip to navigating every meal, ride, and sign along the way.**

### 用户画像更新
| 用户 | 场景 |
|------|------|
| 首次来华游客 | 完全不会中文，需要从签证到点菜的全程支持 |
| 商务差旅 | 短期出差，需要快速交通 + 餐厅 + 翻译 |
| 深度旅行者 | 已确定行程，需要路上实时帮助 |
| 在华外国人 | 日常生活需要的翻译 + 团购 + 出行 |

### 核心价值主张
1. **从规划到路上，一个工具搞定** — 不跳 App，不切换工具
2. **外国人专属优化** — 知道哪些酒店收外宾、哪些餐厅有英文菜单、怎么用支付宝
3. **翻译内置** — 拍照翻译菜单、语音翻译对话，不用再开 Google Translate
4. **实时管家的感觉** — 行程确认后，自动根据时间/位置推荐附近该做的事
