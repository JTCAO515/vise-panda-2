# 0006 — Visa Kit MVP Scope

**Status:** Accepted
**Date:** 2026-06-19

## Context

来中国旅行最大的实际障碍之一是签证。用户需要：
1. 知道自己是否需要签证
2. 准备签证申请材料（行程单、酒店预订单）

全量覆盖所有国家签证政策的成本极高，且政策频繁变动。需要在可控范围内先验证需求。

## Decision

**MVP 范围**：仅覆盖 4 国护照 + 申根区

- 美国护照 → 2024 新规：10 年多次签证需 EVUS 登记
- 英国护照 → 电子签证（需提前申请）
- 澳大利亚护照 → 电子签证
- 加拿大护照 → 电子签证
- 申根国家护照 → 15 天免签（2024 新规试点）
- 其他国籍 → 仅提示「建议联系最近的中国使馆」

**功能清单**：
1. 用户输入国籍 + 目的地 + 日期
2. 系统返回签证类型说明 + 预估办理时间
3. 生成标准英文行程单（模板化，使用已有行程数据）
4. 可预览 + 复制/导出纯文本

**技术实现**：
- 签证政策数据：硬编码 JSON 文件 `data/visa_policies.json`（静态，手动更新）
- 行程单生成：模板字符串拼接，不依赖第三方
- 导出功能：纯前端 `navigator.clipboard.writeText()` 复制

## Consequences

- ✅ 4 国 + 申根覆盖绝大多数 VP-Hermes 目标用户群（英语国家 + 欧洲）
- ✅ 静态数据，无政策同步负担
- ✅ 零外部依赖
- ⚠️ 政策变化时需手动更新 JSON
- ⚠️ 非覆盖国家的用户体验较差（仅提示联系使馆，无实质帮助）
