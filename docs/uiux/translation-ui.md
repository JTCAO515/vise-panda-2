# 翻译面板 UI 规范

> 翻译功能的前端设计（Phase 1.5 核心功能）

## 入口

- 底部导航新增 **Translate Tab**（翻译图标 🌐）
- 点击进入翻译面板

## 面板布局

```
┌─────────────────────────┐
│  [语音] [文字] [短语库]   │  ← 子 Tab 切换
├─────────────────────────┤
│                         │
│    当前模式内容区         │
│                         │
├─────────────────────────┤
│  [麦克风按钮] [输入框]    │  ← 操作区
└─────────────────────────┘
```

## 三种模式

### 1. 语音翻译

```
┌─────────────────────────┐
│                         │
│      🎤                 │
│   按住说话               │
│                         │
│  ┌───────────────────┐  │
│  │ English → 中文     │  │  ← 翻译结果显示
│  │ "How much?" →      │  │
│  │ "多少钱？"          │  │
│  └───────────────────┘  │
│                         │
│  [English] [中文]       │  ← 语言方向切换
└─────────────────────────┘
```

- 按住麦克风按钮录音 → 释放后自动识别+翻译
- 翻译结果双语显示
- 点击喇叭图标播放翻译结果
- 快速语言切换：英→中 / 中→英

### 2. 文字翻译

```
┌─────────────────────────┐
│  ┌───────────────────┐  │
│  │ 输入英文或中文...    │  │  ← 输入框
│  └───────────────────┘  │
│  [翻译 →]               │  ← 按钮
│                         │
│  ┌───────────────────┐  │
│  │ 翻译结果            │  │  ← 结果区
│  │ "..."              │  │
│  └───────────────────┘  │
│                         │
│  📋 复制  🔊 朗读       │  ← 操作
│                         │
│  ── 最近翻译 ──         │
│  "Hello" → "你好"       │  ← 历史（本地）
│  "Thank you" → "谢谢"   │
└─────────────────────────┘
```

- 实时翻译（输入时自动检测语言）
- 翻译历史存储在 localStorage
- "用后即焚"开关（默认关闭）

### 3. 短语库

```
┌─────────────────────────┐
│  🍽️ 餐饮  🚕 打车       │  ← 分类 Tab
│  🏨 酒店  🛍️ 购物       │
│  🆘 紧急  🗺️ 问路       │
├─────────────────────────┤
│                         │
│  宫保鸡丁               │
│  Kung Pao Chicken       │  ← 中英对照
│  🎧 gōng bǎo jī dīng    │  ← 拼音 + 发音
│                         │
│  点这个 / I'll have this│  ← 可用句子
│  不要辣 / No spice      │
│  买单 / Check please    │
│  📋  🔊                 │  ← 复制、发音
│                         │
└─────────────────────────┘
```

- 分类浏览：餐饮、打车、酒店、购物、紧急、问路
- 每项含：中文 + 英文 + 拼音 + 语音
- 常用句子一键复制 / 发音

## 数据流

```
用户输入/语音 → STT(ASR) → 翻译API → 结果展示
                                   → TTS语音播放
                                   → 本地历史存储
```

## 约束

- 语音识别：浏览器原生 Web Speech API 或后端 ASR
- 翻译：后端翻译 API
- 数据不做长期云端存储（除了必要缓存）
- 基础短语包内置离线可用
