# 响应式规则

> 断点、布局适配

## 断点

| 断点 | 宽度 | 目标 |
|------|------|------|
| 移动端 | `< 560px` | 主力设计面 |
| 平板 | `560px - 900px` | 适配过渡 |
| 桌面 | `> 900px` | 宽屏优化 |

## 移动端 ( < 560px )

- 底部固定导航
- 卡片单列
- 内容区填满宽度
- Hero 区紧凑
- Chat 区全屏

## 桌面 ( > 900px )

- 顶部 Tab 栏
- 卡片多列(grid)
- 内容区最大宽度限制
- Hero 区居中宽屏

## 规范

- 使用 `clamp()` 函数做响应式尺寸
- 不用固定的 `px` 宽度
- 移动端 sides: `14px`
- 桌面 sides: `clamp(18px, 4vw, 48px)`
- 安全区域：`env(safe-area-inset-bottom)`
